1.分析需求
    根据输入的ID，查询数据并将对应的数据的Age+1


2.根据需求确定Sql语句
    select * from T_Sudent where Nid=@id
    update T_Student set Age=Age+1 where Nid=@id

3.编写数据访问层中执行该sql语句的方法
    在编写方法前，确定该方法的参数与返回值
    参数：执行sql语句的时候需要外界传入的值
    返回值：该sql语句在数据库中执行完毕后数据库返回什么值，就将当前的方法的返回值设置为对应的类型

4.编写Model层实体模型类
	如果需要，按照数据库中表的列名和列属性建立Model
	没有特殊情况，表名和Model类名相同，列名和属性名相同，列属性必须和属性类型对应

5.编写业务逻辑层方法
    根据表现层的需求编写方法
    如何确定参数：根据表现层采集到的数据，确定参数的个数与类型
    如何确定返回值：根据表现层需要的返回结果确定方法的返回值

6.表现层直接调业务逻辑层方法
	并且对业务逻辑层返回的值进行修饰和整理
